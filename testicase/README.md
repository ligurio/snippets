# testicase

handy tool to pass manual testcases.

- Consume testcases described with [Gherkin](https://docs.cucumber.io/gherkin) syntax
- Produce JUnit and TestAnythingProtocol compliant output

### Usage

```
$ go build
$ ./testicase -file sample.feature
```
