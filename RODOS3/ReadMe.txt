Установка:
1. cd /path/to/dir
2. ./install.sh

Использование:
1. Подключите RODOS-3 к USB
2. sudo ./RODOS3 [параметры]

Используйте параметры
	--help, -h - вызов справки
	--id ID    - обращение к RODOS-3 с соответствующим ID
	--all, -a  - для выполнения на всех RODOS-3
	--on, -n   - включить нагрузку
	--off, -f  - отключить нагрузку
	--read, -r - считать текущее состояние реле
	или запустите без параметров для получения списка всех устройств и их ID
